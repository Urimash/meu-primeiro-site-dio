O que é HTML?

O HTML é uma linguagem de marcação utilizada para criar páginas web.

O que são Tags?

As tags do HTML são elementos de marcação que indicam ao navegador como exibir o conteúdo na página. Abaixo, vou explicar algumas das principais tags do HTML:

  • <html>: define o início e o fim do documento HTML.

  • >head>: define informações sobre a página, como o título, a descrição e palavras-chave.

  • >title>: define o título da página, que é exibido na guia do navegador.

  • >body>: define o conteúdo principal da página, como textos, imagens, links, entre outros elementos.

  • >h1> a >h6>: define os seis tamanhos de cabeçalho para os títulos da página, do maior ao menor.

  • >p>: define um parágrafo de texto.

  • >img>: insere uma imagem na página.

  • >a>: define um link para outra página ou recurso na web.

  • >ul> e >ol>: define listas não ordenadas e ordenadas, respectivamente.

  • >li>: define um item de uma lista.
